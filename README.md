# ML Fundamentals Projects

This repository contains **Jupyter notebooks demonstrating foundational machine learning concepts, algorithms, and exploratory data analysis** using Python, NumPy, and Pandas. It showcases practical applications of classical ML algorithms on real-world datasets.

## Contents

- **Linear Regression** – Regression analysis, gradient descent optimization, Mean Squared Error loss  
- **Logistic Regression** – Classification, cross-entropy loss, train-test evaluation  
- **K-Nearest Neighbors (KNN)** – Classification with distance metric analysis  
- **Naive Bayes** – Categorical and text classification  
- **Support Vector Machines (SVM)** – Non-linear decision boundaries and kernel methods  
- **Exploratory Data Analysis (EDA)** – Data wrangling, cleaning, and visualization using Pandas and NumPy  

## Purpose / Learning Objectives

- Apply classical ML algorithms on real-world datasets  
- Understand and explain loss functions and optimization techniques  
- Perform comparative experiments and analyze algorithm performance  
- Implement clean, modular, and reusable code  
- Practice feature engineering, data preprocessing, and dimensionality reduction
